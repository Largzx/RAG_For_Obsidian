# RAG FOR Obsidian
## 想法来源
- 当我们想得到一些疑惑或者问题的解决方法时，自己的知识库是经过筛选过的符合自己认知与想法，可以很好接受和实践的知识来源，所以现从中提取与问题相关的方案是切实可行的，也能让自己可以回顾笔记内容尤其是技术以外的关于软实力等书籍所提供给我们的经验
## 具体介绍
本插件使用大模型API方式，通过对本地文件的嵌入和本地保存，以及大模型的检索，实现根据知识库回答问题。

## 安装
- 在.opsidain的plugins中创建文件夹rag-kn
- 将三个主要文件放入文件夹中
- 打开obsidian设置，在第三方插件中启用“RAG 知识检索”插件
- 配置相应参数即可

### 具体操作以及功能实现
- 首先要填写API的接口地址以及API密钥（阿里云、openAI、deepSeek等）
- 填写嵌入模型和语言模型的名称
- 选择需要进行嵌入的文件的文件夹
- 点击`重建索引`对所选文件夹中的文件进行嵌入，并会保存为data.json保存为本地文件
---
- 就可以在左边工具栏点击搜索样图标打开
- 在一次`重建索引`后，该插件也实现了**根据文件的修改，及时重新嵌入相关的文件，而不是全部进行重新嵌入**
